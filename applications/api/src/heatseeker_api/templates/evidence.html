Evidence

What this page is: the raw material — every document the system has ever fetched or been given, kept unchanged so every fact can be traced back to its source. You rarely need to act here.
  • Evidence arrives on its own: the worker collects from active sources, and dataset imports store their files here too.
  • Click a row when you want to answer “where did this fact come from?” — the detail page shows the original bytes, extracted text, and access policy.
  • Nothing to see yet? Activate sources or import a dataset, or press Advance pipeline on the dashboard.

Immutable raw documents, content-addressed by SHA-256. Every retrieval records the access policy it was collected under. Re-fetches of identical content bump the seen-count instead of duplicating. PDF, Office, text, and image evidence is signature-detected and processed into versioned derived artifacts.

OCR: {{ ocr_status }} · semantic vision: {{ vision_status }}. Native extraction and image metadata continue without them; machine interpretation is never presented as a confirmed fact.
